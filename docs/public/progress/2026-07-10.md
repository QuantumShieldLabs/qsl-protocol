# QSL Progress — 2026-07-10

This entry is an engineering-evidence summary of the Suite-2 cryptographic-core
completion arc and the docs truth-up that records it. It is not a release
certificate. It makes no public-readiness, production-readiness, post-quantum,
post-compromise, proven-Triple-Ratchet, crypto-complete, bug-free, or
vulnerability-free claim.

## Cryptographic-core completion (NA-0619..NA-0628)

Across a sequence of merged lanes, the Suite-2 hybrid key schedule reached a
correctness-complete state, and its composition was independently analyzed:

- **Single-root DH+PQ composition (NA-0626).** The classical DH ratchet and the
  post-quantum SCKA reseed were unified onto one session root, with a combined
  DH+PQ boundary.
- **CI-gated symbolic analysis (NA-0627, ENG-0028).** A ProVerif 2.05 model of the
  composition *as shipped* proves, under stated abstractions, message-key secrecy
  and injective transcript agreement against an active Dolev-Yao adversary;
  bidirectional healing (post-reseed and post-combined-boundary traffic stays
  secret when all classical DH secrets are compromised, and post-DH-boundary
  traffic stays secret when the ML-KEM decapsulation key is compromised); and
  control-plane advertisement unforgeability. The gate is mutation-tested and its
  first assertion is a tool sanity pair, so it cannot pass on a model that
  compromised nothing. Reviewer-facing record: `docs/design/DOC-G4-002`.
- **Non-contributory-DH guard (NA-0628, ENG-0034, D-1251/D-1252).** Every live
  X25519 DH output is now checked for the all-zero (non-contributory) value that
  RFC 7748 §6.1 requires, and fails closed with a distinct reason code
  (`REJECT_S2_DH_NONCONTRIBUTORY`). Additive negative conformance vectors run on a
  required CI check, and a byte-scan proves the existing vector set unchanged.

## What this does and does not establish

These are symbolic (over abstracted primitives) and code-level results. The
project's claim boundary is unchanged by them: post-compromise / Triple-Ratchet /
production language remains gated on **independent human review**, the ENG-0035
formal follow-up (a multi-epoch unrolling ProVerif does not currently terminate
on), and the A1–A8 abstraction table recorded in `DOC-G4-002`. QSL remains a
research-stage, non-production effort.
